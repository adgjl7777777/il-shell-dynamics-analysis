# rdf_plotter

Plot radial distribution functions (RDF) g(r) for Li⁺–anion pairs (Fig 2).

## Files

| File | Purpose |
|------|---------|
| `rdf_plotter.py` | Read rdf.txt data and plot g(r) for all anions/temperatures |

## Usage

```bash
python rdf_plotter.py
```

Input: `{DATA_ROOT}/{anion}/{T}/rdf.txt`  
Output: Fig 2 (RDF plots)
